# The Lash Haus - Separate Client and Admin Sites

This ZIP contains two separate sites:

## 1. Client Site
Folder: `client-site`

Files:
- `index.html`
- `client.css`
- `client.js`

This is the public website customers see.

## 2. Admin Site
Folder: `admin-site`

Files:
- `admin.html`
- `admin.css`
- `admin.js`

This is the admin dashboard prototype.

Admin login:
- Email: `admin@thelashhaus.co.uk`
- Password: `admin123`

## How to use on GitHub Pages

Upload everything in this ZIP to your GitHub repository.

Your links will be:

- Main landing page: `/index.html`
- Client website: `/client-site/index.html`
- Admin dashboard: `/admin-site/admin.html`

## Important

This is a static GitHub Pages prototype. The admin login is only front-end protection and is not secure for real customer data. For real secure login, shared bookings across devices, payments, and email notifications, you need a backend/database.

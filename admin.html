How this prototype works

The client website and admin dashboard are two separate HTML files. They share the same browser storage, so bookings/contact messages entered on the client site appear here, and admin edits update the client site.

For a real live platform, this local storage layer would be replaced with Supabase database, Supabase Auth, and Stripe webhooks.
